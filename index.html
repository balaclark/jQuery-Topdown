jQuery Topdown

Simple & flexible notification system for jQuery. Hover over the message to stop it from hiding, click to hide immediately.

Options

Here are the options in full with the default values

  • type: "notification"
  • autoWidth: false
  • speed: 200
  • life: 2000
  • sticky: false
  • url: false
  • container_id: "td_container"
  • onClose: false
  • color: false
  • easeIn: "swing"
  • easeOut: "swing"

Topdown can also convert rendered html elements into a notification, see example.

Known issues

  • onClose callback function lags if notification closed by another one opening
  • In IE 7 & less there is a small gap to the right if any body padding is set.
  • Doesn't work in IE 5.5 & less
  • No rounded corners / drop shadows in autoWidth mode in IE

Examples

Default configuration

$.topdown("normal notification");

Topdown can handle as much text as you like:

type

Topdown comes with three default themes, custom themes can be created in CSS and the class name passed here. 

$.topdown("notification", { type: "notification" });

Instead of CSS theming, custom solid colours can be passed: 

$.topdown("custom colour =)", { color: "#bbcdf2" });

sticky

Sticky notifications don't disappear, they stick around until they are clicked on. 

$.topdown("normal notification", { sticky: true });

autoWidth

Notifications don't have to be 100% width. 

$.topdown("this one isn't 100% width", { autoWidth: true });

url

In URL mode, the notification bar becomes a clickable link. 

$.topdown("click to go to google.com", { url: "http://www.google.com" });

onClose

Run a custom function when the notification is closed. This can also be used to chain notifications by passing another $.topdown call into the onClose event. 

$.topdown("this one has an onClose event", { 
	onClose: function() { 
		alert("callback"); 
	} 
});

Easing

Topdown supports the extra easing modes offered by the jQuery Easing plugin & jQuery UI 

$.topdown("custom easing via the easing plugin / jquery ui", {
	easeIn: "easeOutBounce",
	easeOut: "easeOutBounce",
	speed: 800
});

Use embedded HTML

Want to use text / HTML that is already in the DOM? You can use the jQuery syntax you are used to. If you want to strip out any html formatting and just keep the text, pass in the "html: false" option. 

$("#htmlmsg").topdown();

This is some formatted text.
